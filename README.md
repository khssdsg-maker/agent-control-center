## 功能特性

### 智能扫描
- 自动检测电脑已安装的 AI Agent
- 支持 10 种 Agent：MiMo Code、Claude Code、OpenAI Codex、Coffee CLI、豆包、Kimi、千问、腾讯元宝、龙虾 (Claw)、Antigravity
- 实时显示安装状态

### 状态监控
- CPU 使用率监控
- 内存占用监控
- 磁盘空间占用
- 系统资源总览

### 技能管理
- 每个 Agent 独立的技能展示
- 读取真实技能数据（Claude Code、MiMo Code）
- 技能中文翻译
- 悬停提示查看英文原文

### Agent 调用终端
- 在应用内直接调用 Agent
- 支持 CLI 和桌面应用两种类型

### 聊天记录
- 扫描 Claude Code、MiMo Code、Kimi 的聊天记录
- 按 Agent 分类查看

### 任务系统
- 创建、执行、跟踪任务
- 任务优先级设置
- 任务搜索和筛选

### 工作流编排
- 多步骤任务编排
- 步骤间依赖关系
- 工作流执行和状态跟踪

### 数据分析
- CPU/内存使用率图表
- 任务趋势可视化
- 任务完成率统计

### 多主题
- 7种主题颜色
- 自定义壁纸背景

### 多语言
- 8种语言支持：中、英、日、韩、法、德、西、俄

### 设置
- 主题切换
- 语言切换
- 通知开关
- Agent 图标自定义
- Agent 配置
- 数据导出/导入

### 全局搜索
- Ctrl+F 快捷键打开
- 搜索 Agent 和技能

| 技术 | 说明 |
|------|------|
| Electron | 桌面应用框架 |

| TypeScript | 类型安全 |

## 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建打包
```bash
npm run build
npm run pack
```

## 项目结构

```
agent-control-center/
├── electron/                    # Electron 主进程
│   ├── main.ts                 # 主进程入口
│   ├── preload.ts              # 预加载脚本
│   ├── scanner/                # Agent 扫描器
│   ├── store.ts                # 数据存储
│   └── notifier.ts             # 通知系统
├── src/renderer/                # React 前端
│   ├── components/             # UI 组件
│   ├── pages/                  # 页面
│   ├── stores/                 # 状态管理
│   └── lib/                    # 工具库
├── package.json
└── README.md
```
